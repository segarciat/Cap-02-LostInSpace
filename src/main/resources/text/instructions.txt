objectives: check game winning objectives

look room <right click>: inspects the current room you're in for items, interactable objects, and exits

inspect item -or- inspect object: receive a description of what was inspected, look inside of containers
<the inspect command will often reveal details about something you are confused about>

go direction: use mouse to <left click> provided directions
<directions: go North,  go South, go East, go West>

use item <left click> : use an item in your inventory or an object in the same room as you
<not all items can be used at all times or in every room. Experiment with your options!>

restart: restart the game

exit <escape to  open "Help Menu" to review help instructions > : quits the current game.

menu : use escape key that provides the "Help Menu" that displays the instructions

easymode: too hard? Use this command to never use oxygen when moving.
<use this command again to turn this feature off>
